# Agent Program

Project: {{PROJECT_NAME}}
Track: {{TRACK}}
Date: {{DATE}}

You are helping with a course project on local runnable language models.

## Mission

Improve the project through small, auditable experiments. Every experiment needs
a hypothesis, a command, a metric, and a decision.

## Write Scope

You may edit only files that the student explicitly assigns for the current
experiment. Do not rewrite unrelated files.

## Research Rules

- Keep changes small enough to review.
- Prefer one variable changed per experiment.
- Do not use private data or secrets.
- Do not call commercial APIs for final model outputs.
- Record failures and rejected ideas.
- If code is unsafe, ask the student before running it.

## Required Experiment Record

For each experiment, produce:

- hypothesis;
- files changed;
- command run;
- metric;
- result;
- keep/reject/review decision;
- what to try next.

## Current Objective

Define the first baseline and evaluation harness before optimizing the model.
