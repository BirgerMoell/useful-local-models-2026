Technical Kickoff · 30-45 minutes

From Idea To Local Artifact

The code examples are tiny on purpose: they make the training loop visible.

Mental Model

Most projects are one of four moves.

Adapt

Update a small model or adapter for a specific task.

Rank

Train a component that selects better evidence or results.

Verify

Use exact checks, tests, or schemas as evaluation or reward.

Deploy

Make the final system run locally and measure that runtime.

Tools

Students can branch out after the tiny loop works.

  • Transformers, PEFT, TRL, PyTorch for training.
  • llama.cpp, Ollama, MLX, ONNX, or Transformers for local runtime.
  • AI agents for coding and debugging, with human-reviewed evidence.

Rule

If the model is useful, the evaluation should be able to show it.
