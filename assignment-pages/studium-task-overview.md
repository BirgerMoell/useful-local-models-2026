# Studium Page: Task Overview

The course follows a research-paper cycle. You will read and discuss papers,
write a project proposal, carry out an individual project, submit a first paper,
review two peer papers, revise your work, present it at the final workshop, and
submit a final report.

For the Useful Local Models theme, each project must also include:

- a local use case;
- a baseline;
- a model or component whose parameters are trained or adapted;
- held-out evaluation;
- ethical considerations;
- a local runnable or inspectable artifact.

The example code and demos are starting points. Your submitted project should
have its own research question, dataset, method choice, and evaluation design.

## Assignments

| Assignment | Main purpose | Weight / status |
|---|---|---|
| Philosophy assignment | Science theory and philosophy of science. | 15% |
| Literature seminars | Present and discuss research papers. | 15% |
| Project proposal | Scientific proposal, popular science abstract, and proposal presentation. | 15% |
| Progress seminars | Supervision, reporting, and project development. | obligatory |
| First report | Complete first version of the term paper for review. | required for review |
| Peer reviews | Review two peer papers. | 15% |
| Project presentation | Present the final project at the workshop. | part of final work |
| Final report | Revised scientific paper and artifact documentation. | 40% |

Progress seminars are obligatory and are the main supervision setting. If you
miss a seminar, contact your supervisor promptly for a compensation task.
