Proposal Clinic · 20-30 minutes

Make The Project Smaller

A good proposal makes the evaluation possible.

Proposal Test

Can the project be answered by one result table?

If not, the research question is probably too large.

Required Decisions

TaskWhat exact input becomes what exact output?
DataWhere do examples come from, and what is the split?
BaselineWhat simple method would be embarrassing to lose to?
TrainingWhat parameters change, and why?
MetricWhat does useful mean numerically and qualitatively?

Bad Scope

Train a Swedish assistant.

  • Too broad.
  • No clear baseline.
  • No obvious held-out test.

Better Scope

Extract action items from Swedish support emails.

  • Specific input and output.
  • Rule-based baseline is feasible.
  • Held-out exact-match and error analysis are meaningful.

Fallback

Every proposal needs a smaller version that still teaches something.
