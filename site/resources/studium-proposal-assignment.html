Useful Local Models

Studium Assignment Text: Project Proposal

Submit your scientific project proposal as a PDF.

Your proposal should follow the normal course instructions for project proposals

and include the following local-model sections.

Required Theme-Specific Content

Local Use Case

Describe what the local model is supposed to do and why it is useful to run it

locally rather than through an external API.

Model And Training Plan

Specify:

  • base model or model family;
  • training method;
  • what parameters will be updated;
  • expected compute;
  • fallback plan if the method is too expensive or unstable.

Data Plan

Specify:

  • data sources;
  • license or access conditions;
  • train/dev/test split;
  • possible contamination or leakage risks;
  • whether the data can be redistributed.

Baseline And Evaluation

Specify:

  • baseline system;
  • evaluation dataset;
  • metrics;
  • error analysis plan;
  • significance testing plan, if applicable.

Local Deployment Plan

Specify:

  • expected final artifact format;
  • local runtime;
  • target hardware;
  • how another person will run the model.

Ethical Considerations

Discuss relevant risks such as privacy, copyright, bias, environmental cost,

misuse, and overclaiming.
